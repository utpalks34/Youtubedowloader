# YouTube Video Downloader

This is a simple YouTube Video Downloader built using Python.  
You just need to paste the YouTube link, and the app will download the video in the highest quality.

I made this project to practice Tkinter and basic GUI development in Python.



#Features
- Clean and simple user interface
- Paste any YouTube link and download the video
- Downloads in the highest available resolution
- Lets you choose where to save the file
- Runs download process in a separate thread (UI does not freeze)



#tech
- Python
- Tkinter (GUI)
- Pytube (YouTube downloader library)
